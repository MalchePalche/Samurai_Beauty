# Samurai Beauty

A full website for **Samurai Beauty** — a luxury beauty studio run by Tina, a certified Pro Makeup Artist and Master Esthetician based in New York. The brand is rooted in Japanese precision, calm, and craft.

## Pages

| Page | Description |
|---|---|
| `index.html` | Home — hero, services teaser, stats, quote, gallery preview |
| `about.html` | About Tina — philosophy, bio, brand partners (MAKE UP FOR EVER, NARS) |
| `services.html` | Full services list — Makeup, Lashes, Skin Rituals, Brow Design |
| `gallery.html` | Filterable image gallery — All / Makeup / Lashes / Skin / Editorial |
| `contact.html` | Contact form with service inquiry dropdown |
| `booking.html` | Multi-step booking flow — Ritual → Service → Date & Time → Details |

## Tech

- Vanilla HTML / CSS / JavaScript — no frameworks, no build step required
- Fonts: [Shippori Mincho](https://fonts.google.com/specimen/Shippori+Mincho) (display) + [Jost](https://fonts.google.com/specimen/Jost) (body) via Google Fonts
- Images: [LoremFlickr](https://loremflickr.com/) for tagged beauty placeholder photos
- CSS custom properties design system (`--parchment`, `--ink`, `--blossom`, `--accent`, etc.)
- Scroll-reveal animation system (`.reveal`, `.reveal-stagger`, `.reveal-left`, `.reveal-right`)
- `prefers-reduced-motion` support throughout

## Running Locally

No build step needed. Just open any `.html` file in a browser, or serve with any static server:

```bash
npx serve .
```
